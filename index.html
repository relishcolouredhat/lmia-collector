üåç Geographic Data Enhancement

We automatically enhance the data with geographic coordinates by:

  • Extracting postal codes from address fields using pattern matching
  • Converting to coordinates using OpenStreetMap's Nominatim geocoding service
  • Adding lat/lon columns to enable mapping and geographic analysis

This enhancement adds valuable location context while preserving all original government data exactly as provided.
